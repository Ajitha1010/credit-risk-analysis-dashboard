# Credit Risk & Default Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes customer credit default behavior using demographic, payment history, and billing data.

The dashboard helps identify high-risk customer segments and repayment patterns associated with default risk.

---

## Tools & Technologies
- Power BI
- DAX
- Data Modeling
- Data Visualization

---

## Dashboard Pages

### 1. Executive Summary
- Total customers
- Default rate
- Average credit limit
- Default distribution by gender and education

### 2. Age Risk Analysis
- Defaults by age group
- Customer distribution by age
- Gender-based age risk analysis

### 3. Payment Behavior Analysis
- Repayment delay analysis
- Average bill vs payment amount
- Payment behavior patterns linked to default risk

---

## Key Insights
- Approximately 23% of customers defaulted on payments.
- Customers aged 25–34 showed the highest default counts.
- Repayment delays strongly correlated with higher default risk.
- University-level customers contributed the highest number of defaults.
